vagrant_howto.txt
----------------------
Install the latest version of virtualbox:
	- https://www.virtualbox.org/wiki/Downloads

Install vagrant 1.4.3:
	- http://www.vagrantup.com/download-archive/v1.4.3.html
	- 1.4.3 is used instead of the latest 1.5x because a dependency, vagrant-berkshelf, is not available for 1.5.
	- http://www.vagrantup.com/downloads.html
	- vagrant plugin install vagrant-berkshelf --plugin-version 2.0.0.rc2

Install ruby for windows 1.9.3 and the devkit:
	- http://rubyinstaller.org/downloads/
	- Extract the devkit under the ruby install folder (like C:\Ruby193) in a directory named something like "devkit".
	- Cd to the directory, and run "ruby dk.rb init"
	- Next, run "ruby dk.rb install".

?? Install the latest version of chef:
	- http://www.opscode.com/chef/install.msi

Install berkshelf for vagrant:
	- gem install berkshelf
	- vagrant plugin install vagrant-berkshelf
	- vagrant plugin install vagrant-hostmanager
